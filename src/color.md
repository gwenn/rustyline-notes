# Color

## Environment variables

- `NO_COLOR`
- `COLORTERM`
- `TERM`

```
TERM=xterm-256color

COLORTERM=truecolor
TERM=xterm-kitty

COLORTERM=truecolor
TERM=xterm-256color

TERM=linux
```
